== Description ==

**Simply create polls and embed them in your website easily.**

= Instructions =
After activation of the plugin, visit the "**Polls**" menu page and create a poll. A **unique shortcode** will be assigned to the poll so you can
embed it in post's,page's or anywhere on your site.

== Installation ==

1. Download and unzip the polls plugins.
2. Upload to your wp-content/plugins folder
3. Activate from the dashboard
4. Go to the new Polls menu page and create a poll.
5. A new shortcode will be given to you.
